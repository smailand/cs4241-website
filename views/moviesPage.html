"Technical Achievement" Work - Assignment 3:

For the final 10% of the grade basd on "technical achievement" I added some client side javascript.
This javascript uses the input in the text box to compose a search query that will be used to send a get request to the server.
The server then uses that search query to filter the list of movies provided for the project and sends back that filtered list
The client then gets that response and updates the HTML to reflect that filtered list. (No filtering is done on the client).
The client side javascript also modifies the search query in the URL to reflect what was searched so that the link can be given to others.
The text box is also autopopulated based on what is in the search query portion of the URL.
